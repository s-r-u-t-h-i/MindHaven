🧠 **MindHaven - an AI-Driven Psychological Disorder Detection App**

This project is a full-stack MERN-based mobile application designed to support the automated detection of psychological disorders and provide intelligent treatment recommendations using AI-based EEG analysis.

The application integrates a mobile-friendly MERN stack platform with a Deep Neural Network (DNN) model that analyzes pre-processed EEG-derived features such as emotional regulation, stress response, cognitive load, and attentional control. By leveraging objective neurophysiological data instead of subjective clinical assessments, the system enables early, scalable, and remote psychological evaluation.

🔹 **Key Features**

- AI-based psychological disorder detection using EEG features

- Deep learning-driven multi-class classification

- Personalized treatment recommendations

- Remote accessibility and continuous monitoring

- Secure authentication with login and logout

- User dashboard for viewing diagnosis results

- Diagnose tab for submitting EEG data and receiving predictions

- Profile tab for managing user information and history

🔹 **Technologies Used**

- **Frontend:** React Native (Expo)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI/ML:** Deep Neural Networks (DNN), EEG feature analysis
- **Deployment:** Render
- **Integration:** RESTful APIs

🔹 **Purpose**

The project aims to combine artificial intelligence and modern full-stack development to deliver an objective, accessible, and scalable solution for mental health assessment, supporting early detection and informed decision-making in psychological care.
